Predefined constants

The following constants are predefined inside the PHPLint parser:

/*. boolean .*/ FALSE
/*. boolean .*/ TRUE
Logical values. PHPLint accepts the lower-case letters version false and true.

/*. float .*/ NAN
Not-A-Number.

/*. float .*/ INF
Infinity.

/*. null .*/ NULL
This constant is the only value allowed for the type null and it can match a string, an array, a resource, an object or a mixed. PHPLint also accepts the lower-case letters version null.

/*. resource .*/ STDIN
/*. resource .*/ STDOUT
/*. resource .*/ STDERR
These constant are special the sense that constants of type resource cannot be defined in user code (only scalar values and strings are allowed). Note too that these constants are defined only in the CLI version of PHP, but this cannot be verified by PHPLint.

The following magic constants are not real constants, but special expressions of the language whose value depends on the context in which they appear. PHPLint calculate their value and substitute this value in place according to the specifications of the PHP manual: 

/*. string .*/ __DIR__         (PHP 5 only)
/*. string .*/ __FILE__
/*. int    .*/ __LINE__
/*. string .*/ __NAMESPACE__   (PHP 5 only)
/*. string .*/ __FUNCTION__    (lowercase in PHP 4)
/*. string .*/ __CLASS__       (lowercase in PHP 4)
/*. string .*/ __METHOD__      (PHP 5 only)
